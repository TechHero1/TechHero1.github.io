Interactive Furniture (mobílias interativas) aumentam as chances de aparição de uma estudante específica no café.
Colocar mais de uma Interactive Furniture no mesmo café acumula as chances, mas a porcentagem adicionada é relativamente menor. A partir da quinta, a chance para de acumular.[carece de fontes]
Remover Interactive Furniture de outras estudantes não desejadas.
Se tiver 2 presentes de efeito pequeno (), vale mais a pena usá-los para craftar gift box de 1 presente favorito ().
A possibilidade de tocar nas estudantes reseta a cada 3 horas, podendo aumentar o XP múltiplas vezes antes que as visitas mudem.
